# 📘 MINI PROJECT REPORT: A.U.R.A.

**A.U.R.A. (AI Assistive User Reminder Application)**

A.U.R.A. is an intelligent, affordable assistive system designed to help dementia patients manage daily life while reducing the burden on caregivers. It combines real-time face recognition and conversational AI to provide a culturally sensitive and accessible care solution.

## 📁 Project Structure

The LaTeX report is organized as follows:

* 📄 `thesis.tex`: The master LaTeX file.
* 📂 `Frontmatter/`: Contains the front page, formatting rules, and abstract.
* 📂 `Chapters/`: Contains the core content (Introduction, Literature Survey, etc.).
* 📂 `Backmatter/`: Contains references and annexures.
* 🖼️ `Image/`: Stores all diagrams and visuals used in the report.

---

## 📄 How to Compile the Report

To generate the PDF version of the mini-project report:

1. ⚙️ Ensure you have a LaTeX distribution installed (e.g., TeX Live, MiKTeX).
2. 💻 Open the project in your favorite LaTeX editor (VS Code with LaTeX Workshop, TeXstudio, etc.).
3. ▶️ Compile the main file:

   ```bash
   latexmk -pdf thesis.tex
   ```

   *Alternatively:* Run `pdflatex thesis.tex` (you may need to run it twice to sync references).
